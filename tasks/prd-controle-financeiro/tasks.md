# Lista de Tarefas: Controle Financeiro Frontend

Esta lista detalha as tarefas de alto nível para o desenvolvimento do frontend do Controle Financeiro, com base no PRD e Tech Spec fornecidos. Cada tarefa representa um entregável funcional e incremental, e inclui a etapa de build e testes unitários relevantes.

## Tarefas Principais

### 1.0 Backend API Enhancements
Implementar novos endpoints e atualizar os existentes no backend NestJS para suportar filtragem, importação/exportação e novos dados de dashboard, incluindo a configuração CORS necessária.

### 2.0 Frontend Setup
Inicializar a aplicação Next.js, configurar estilização com Tailwind CSS, e integrar Shadcn/UI, TanStack Query, e Chart.js.

### 3.0 Core Layout e Theming
Desenvolver o layout principal da aplicação (cabeçalho, barra lateral) e implementar a funcionalidade de alternância de tema (modo claro/escuro).

### 4.0 Categories Module Implementation (Frontend)
Desenvolver a interface de usuário para criar, listar e excluir categorias, integrando-a com a API do backend.

### 5.0 Expenses Module Implementation (Frontend)
Implementar a interface de usuário para listar, filtrar e adicionar despesas, integrando-a com a API do backend.

### 6.0 Dashboard Module Implementation (Frontend)
Criar e integrar componentes de gráfico para relatórios financeiros anuais e mensais.

### 7.0 Import/Export Functionality (Frontend)
Desenvolver a interface de usuário e a lógica para importar e exportar despesas via CSV, incluindo validação no lado do cliente.
